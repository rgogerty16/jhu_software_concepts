Module 1: Personal Flask Website

A personal website built using Flask, a lightweight Python web framework. The site features a homepage with a biography and photo, a contact page with professional links, and a projects page showcasing coursework. The application is structured using Flask Blueprints for modular page routing, Jinja2 HTML templates for dynamic rendering, and custom CSS for layout and styling.

This project demonstrates core web development skills including server-side routing, template inheritance, static file management, and responsive CSS layout — all within a clean, reproducible Python environment.
